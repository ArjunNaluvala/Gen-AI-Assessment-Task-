##  GenAI Prompt Engine using LangChain

##  Project Overview

This project demonstrates how to build a dynamic and reusable prompt system using LangChain instead of hardcoded prompts. It simulates a real-world prompt engineering pipeline where user inputs are validated and transformed into structured prompts.

##  Objective

* Replace hardcoded prompts with reusable templates
* Build scalable prompt pipelines
* Implement input validation
* Design multi-style prompt generation systems

##  Tech Stack

* Python
* LangChain
* Jupyter Notebook

##  Features

* PromptTemplate (No hardcoding)
* Multi-input dynamic prompts
* Prompt variations engine (Teaching, Interview, Storytelling)
* ChatPromptTemplate (role-based system)
* Input validation layer
* Prompt Generator App
* Template reusability

##  Pipeline Flow

User Input → Input Validation → Prompt Template Selection → Dynamic Prompt Generation → Output

##  Key Learnings

* Designing modular prompt systems
* Using LangChain PromptTemplate effectively
* Handling multiple inputs dynamically
* Building scalable GenAI pipelines
* Importance of input validation in LLM systems

##  Example Output

Input:
Topic: Neural Networks
Audience: beginner
Tone: fun

Output:
Explain Neural Networks for beginner in a fun storytelling style

##  Why This Project Matters

This project focuses on engineering reusable prompt systems instead of writing one-time prompts. It reflects how real-world Generative AI applications are designed with scalability, flexibility, and modularity.
